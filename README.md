# ZA Portfolio Site

This repository contains the code for the personal portfolio of **ZA**, a scientist and training‑and‑workflow professional focused on AI‑powered improvements. The site is built as a static website using HTML and CSS. It can be hosted on GitHub Pages or any static site host.

## Structure

- `index.html` – home page with hero section, value tiles, and featured artifacts.
- `artifacts.html` – library listing all artifacts.
- `artifact-planner.html` – details about the Planner Workflow Template.
- `artifact-copilot.html` – details about the Copilot Prompt Library.
- `artifact-sop.html` – details about the SOP for Data Organization &amp; Reproducible Pipelines.
- `artifact-training.html` – details about the sample training outlines.
- `about.html` – about page with biography, skills, and principles.
- `contact.html` – contact page with a simple email form.
- `css/style.css` – styles for the site.
- `downloads/` – placeholder files (Markdown) for each artifact that can be replaced with your own content.

## Running Locally

1. Clone or download this repository.
2. Open `index.html` in your web browser. The site is entirely static, so no server is required.

## Deploying to GitHub Pages

1. Create a new GitHub repository and push the contents of this folder.
2. In the repository settings, enable GitHub Pages for the `main` branch (or `gh-pages` branch if you prefer).
3. GitHub will publish the site at `https://&lt;your-username&gt;.github.io/&lt;repository-name&gt;/`.

## Customizing

- Update the placeholder files in `downloads/` with your own content.
- Modify the HTML pages to change copy or add new artifacts.
- Update colors or typography by editing `css/style.css`.
